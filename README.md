## Overview

A funky Japanese birthday luck calculator that discovers your cosmic birthday energy!

## Features

- Check your birthday luck with a beautiful Japanese-inspired interface
- Cosmic energy calculations based on your birth date
- Responsive design with smooth animations

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Radix UI Components
